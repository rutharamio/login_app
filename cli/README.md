# CLI Scripts (login_app)

Esta carpeta contendrá scripts de línea de comandos (CLI) para tareas backend
que no deben ejecutarse desde el navegador.

Ejemplos futuros:
- Sincronización incremental de Gmail
- Refresco automático de tokens OAuth
- Limpieza de tokens expirados
- Tareas programadas vía cron

Actualmente no se utiliza en producción.
